## Subject
* 사용자는 웹페이지에 접속하여 알람음을 입력.<br/>
ex)“히라야, 일어나 공부해야지.” 
* 웹캠을 통해 사용자의 얼굴을 인식하여 졸음 감지
* 졸음 감지후, 사용자가 입력한 알람음을 TTS를 통해 출력.
* 미리 지정된 제스처 등을 인식하여 알람을 멈춤
* 추가적인 기능으로 조는 모습캡쳐 후 SNS로 공유하기
